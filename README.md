## Installation

To get a local copy up and running, follow these simple steps:

1. **Clone the repo**
    ```sh
    git clone https://github.com/Radhika-Shirke/radhika-shirke-portfolio.git
    ```
2. **Install NPM packages**
    ```sh
    npm install
    ```
